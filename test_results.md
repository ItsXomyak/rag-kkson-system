# Результаты тестирования RAG KKSON

**Дата:** 07.04.2026  
**Система:** RAG-система поиска по научным журналам ККСОН

## Результаты

### Тест 1. Узконаправленный запрос (RU)

**Запрос:** `Закон об арбитраже 2016 года Казахстан`

**Результат:** Система точно нашла информацию о Законе РК «Об арбитраже» от 8 апреля 2016 года (№ 488-V ЗРК) и предоставила контекст:
- Закон объединил два ранее действовавших закона: «О международном коммерческом арбитраже» и «О третейских судах» (2004 г.)
- Определён как четвёртый этап развития арбитражного законодательства
- Указано значение для арбитражного процесса в Казахстане

**Источники:** 2 статьи из журнала kgu-law  
**Оценка:** ✅ **Отлично** — точный, фактологически корректный ответ с конкретными датами и номерами НПА

---

### Тест 2. Широкий запрос (RU)

**Запрос:** `образование в Казахстане`

**Результат:** Система предоставила комплексный обзор, охватывающий:
- Структуру системы образования (3 ступени)
- Болонский процесс и трёхступенчатую модель высшего образования (с 2007 г.)
- Проблемы: недофинансирование, переполненность школ, низкие результаты PISA-2012
- Качество подготовки педагогов и дисбаланс с рынком труда

**Источники:** 4 статьи из разных журналов (филология, педагогика, сельское хозяйство)  
**Оценка:** ✅ **Отлично** — широкое покрытие темы, разнообразие источников подтверждает работу multi-query expansion

## Ключевые выводы

1. **Точность поиска:** узконаправленные запросы находят конкретные НПА с номерами и датами
2. **Широта покрытия:** multi-query expansion обеспечивает разнообразие источников по широким темам (4–5 статей из разных журналов)
3. **Мультиязычность:** bge-m3 корректно работает с RU/KZ/EN — запрос на одном языке находит статьи на других
4. **Отсутствие галлюцинаций:** при отсутствии данных система сообщает об этом без попыток придумать ответ
5. **Качество структуры:** ответы лаконичны, каждое утверждение подкреплено ссылкой на источник

## Архитектура pipeline

```
Пользователь
    │
    ▼
[Query Expansion] ── LLM генерирует 3 подзапроса
    │
    ▼
[Multi-Query Search] ── 4 запроса × 15 чанков + keyword-поиск
    │
    ▼
[Cross-Encoder Reranker] ── оценка релевантности, фильтрация мусора
    │
    ▼
[LLM Generation] ── Qwen 3, ответ на основе top-8 источников
    │
    ▼
Ответ + ссылки на источники
```
